# 1. Image Processing (Grayscale Blur, Edge Detection)

Using only NumPy — no OpenCV.

## Tasks

- Generate a 100×100 random grayscale image (values 0–255).

- Apply 3×3 average blur (same idea as your smoothing filter).

-Apply a Sobel-like edge detection manually using convolution kernels:

    - Horizontal

    - Vertical

- Combine both gradients to get an edge intensity map.

### Skills you reinforce

- Padding

- Multi-slice convolution

- Broadcasting

- Vectorized math
